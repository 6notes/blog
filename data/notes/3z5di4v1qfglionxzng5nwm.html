pushSettings

(WIP) Changing git push settings to be less tedious

  • I think the default git push settings are annoying because you always have to specify the origin or upstream stuff.
    • I think if you set default = current under [push] in your global .gitconfig file, you can just git push to whatever your current repo's branch is. I think it's more convenient and haven't run into any problems yet.
